# Segoe MDL2 Assets 图标码位表(已验证)

⚠️ **重要**:这些码位在 **Photoshop CC 2019 + Windows** 上用画布探针法实测有效。**换字体版本/系统可能失效**——用之前若不确定,先用探针法(见 `scripts-reference.md`)重新验证。

字号 **≥24px**,否则可能渲染成空方框。用法:`String.fromCharCode(0xE9E9)`,字体设 `SegoeMDL2Assets`。

## ✅ 实测有效(放心用)

| 码位 | 图标 | 适用场景 |
|------|------|---------|
| `0xE9E9` | 水平滑块(横线+圆点) | 调度、参数、控制、产线 |
| `0xF22C` | 网络节点 / 关系图 | 物联网、连接、协同、行业网络 |
| `0xF158` | 3D 立方体 | 数字孪生、产品、模型、3D |
| `0xE9D5` | 清单勾选 | 任务、MES、执行、检查 |
| `0xE713` | 齿轮 / 设置 | 内核、引擎、配置、研发 |
| `0xF8B0` | 实心齿轮 | 同上(实心风格) |
| `0xE823` | 历史时钟(带回拨箭头) | 7×24、记录、历史、响应 |
| `0xE81C` | 标准时钟 | 时间、时效 |
| `0xE2AD` | 标准时钟 | 时间、时效(备选) |
| `0xE753` | 云 | 云平台、云服务 |
| `0xE7B7` | 定位针 | 地点、网点、位置 |
| `0xE7F4` | 显示器 / 监控 | 监控、可视化、大屏 |
| `0xE8A9` | 窗口 / 看板 | 看板、仪表盘 |
| `0xE1A5` | 文件上传 | 上传、导入 |
| `0xE9D9` | 数据折线图 | 数据、增长、分析 |
| `0xE809` | 网格面板 | 面板、矩阵 |
| `0xEA37` | 列表 | 列表、目录 |
| `0xE72A` | 右箭头 → | "了解更多/阅读全文"链接尾 |

## ❌ 实测无效(别用,会渲染成空方框)

| 码位 | 本想要 | 实际 |
|------|--------|------|
| `0xE917` | 时钟 | 空方框(改用 `0xE823`/`0xE81C`) |
| `0xE72E` | 盾牌 | 空方框(用 `0xE713` 齿轮替代语义) |
| `0xE964` | 芯片 | 偏位/不清晰 |
| `0xE9F5` | Logo图标 | 渲染成两个小齿轮叠加,不可用 |

## 经验

- **Logo 别硬凑图标字体**——不如直接用中文字(如"智"字)+ 加粗,更清晰专业可控。
- 需要新图标时,一次探针放 8~12 个候选,批量验证效率最高。
- 微软官方 MDL2 码位文档仅供参考起点,**以本机实测为准**。
